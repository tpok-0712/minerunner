2名のプレイヤーがそれぞれゴールを目指しつつ、フィールドに地雷を置いて相手を妨害する対称型対戦ゲームです。
それぞれのプレイヤーはAP(アクションポイント)を使い、移動(1AP)・地雷の設置(2AP)/解除(1AP)・そして移動速度を2ターンの間2倍にできるオーバークロック(3AP)という選択肢があります。
APは2まで次のターンに持ち越すことができます。
地雷は置かれた瞬間自分からも敵からも見えなくなり、敵味方関係なく踏んだら爆発し、初期地点まで戻されます。
その代わり、各プレイヤーは常に自分の半径4マス以内の地雷の個数を知ることができます。
APと地雷を効率よく使い、相手を出し抜きましょう。
